Guía Rápida

Bienvenido a QSO Uploader. Esta aplicación te permite subir automáticamente tus contactos (QSO) a varios Log Online.

Configuración

Configura de la aplicación.

En la sección de Configuración debes de:

  • Configurar tus credenciales de acceso a los diferentes Log Online, usuario y contraseña.
  • Configurar tu Señal Distintiva principal.

Configuración de los softwares:

  • Log4OM
    • Ve a Settings.
    • Luego a Program Configuration.
    • Luego a menu Connections.
    • Rellena los datos de la siguiente manera:
      • Port: 2233
      • Connection name: QSO Uploader
      • Services Type: ADIF_MESSAGE
    • Y le das a signo +
    • Y listo

  • WSJT-X
    • Ve a Archivo
    • Ajustes
    • Luego a Reportes
    • Busca Servidor UDP Secundario y verifica que la IP sea 127.0.0.1 y el Puerto 2333
    • Tilda la opción: Habilita "broadcast" de contacto Guardado (ADIF)
    • Y listo
  • JTDX
    • Ve a Fichero
    • Ajustes
    • Luego a Informes
    • Busca Habilitar envío servidor UDP secundario y tilda
    • Verifica que 2do servidor UDP la IP sea 127.0.0.1 y el Puerto UDP sea 2333
    • Y listo
  • N1MM
    • PROXIMAMENTE

Cómo subir contactos

  1. Selecciona el software de que utilizaras para enviar la información(Log4OM, WSJT-X/JTDX, N1MM).
  2. Al seleccionar el software la aplicación quedara a la escucha de la información en el puerto correspondiente.
  3. Realiza la configuración de Log4OM/WSJT-X/JTDX/N1MM según la guía anterior.
  4. Cada vez que guardes un contacto en alguno de estos softwares, QSO Uploader recibira la información en el puerto correspondiente y la enviara a la plataforma configurada.
  5. Si sale un error, verifica que la configuración sea correcta.

Proximamente se agregara la opción de subir contactos desde un archivo ADIF.

Preguntas Frecuentes

¿Qué formatos de archivo son compatibles?

La aplicación es compatible con archivos ADIF (.adi, .adif) exportados desde la mayoría de los programas de registro de contactos.

¿Cómo configuro el puerto UDP?

La aplicación escucha automáticamente en los puertos correspondientes a cada software. Solo debes de configurar el software que utilizaras para enviar la información(Log4OM, WSJT-X/JTDX, N1MM). Y seleccionas el software en la aplicación.

¿Se piensa en agregar mas funcionalidades?

Si, estoy abierto a sugerencias y sugerencias de nuevas funcionalidades. Si tienes alguna sugerencia, no dudes en contactarme.

¿Se puede usar con otros software que no estan en la lista?

Si, se puede usar con otros software que no estan en la lista. El software debe tener la opcion de envio de información via UDP. Solo debes de configurar el software que utilizaras para enviar la información como alguno de los que estan en la lista y listo. Por ejemplo DXLog.net transmite los datos via UDP en el puerto 12060 como N1MMM, seleccionas N1MM en la aplicacion y escuchara a DXLog.net para subir los QSO.
